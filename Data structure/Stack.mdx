### What is Stack
A **Stack** is a linear data structure that follows a particular order in which the operations are performed. 
The order may be **LIFO(Last In First Out)** or **FILo(First In Last Out)**. LIFO implies that the element that is inserted last,
comes out first and FILO impies that the element that is inserted first, comes out last
### Key Operations on Stack Data Structure
- Push
- Pop
- Peek
- IsEmpty
- IsFull
### Applications of Stack
- Recursion
- Expression Evaluation and Parsing
- Depth-First Search (DFS)
- Undo/Redo Operations
- Browser History
- Function Calls
